# APK files

Drop your release APKs here with these exact names (they are linked from index.html):

- repmotion.apk
- calorix.apk
- qlodi.apk        (then set apk:'assets/apk/qlodi.apk' in index.html)
- cashpilot.apk    (then set apk:'assets/apk/cashpilot.apk' in index.html)

On GitHub Pages the download URL becomes:
  https://<username>.github.io/<repo>/assets/apk/repmotion.apk

Note: GitHub warns on files >50 MB and blocks >100 MB. If an APK is large,
use a GitHub Release instead and point the apk field at:
  https://github.com/<username>/<repo>/releases/latest/download/repmotion.apk
